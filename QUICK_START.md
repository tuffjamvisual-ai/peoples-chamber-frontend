# People's Chamber - Quick Start Guide

## What We're Building
A UK politics voting platform where citizens vote on Parliament bills and compare their votes with how MPs voted.

**Live Site:** https://peoples-chamber-frontend.vercel.app

## What's Working ✅
1. 3,865 UK Parliament bills with complete data
2. User signup/login
3. Voting system (Support/Oppose)
4. Vote counts update instantly
5. Instant page loads (<50ms)

## What Still Needs Building
1. House of Commons vote data (show real MP votes)
2. MPs page (list all MPs and their voting records)  
3. User dashboard (voting history)
4. Mobile responsive design
5. Fix "1000 bills" display bug

## Tech Stack
- Next.js 16 + TypeScript
- Supabase PostgreSQL
- Vercel hosting
